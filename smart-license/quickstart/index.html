快手上手

# 一、云端申请 License

  1. 登录 Shield.盾 (opens new window)
  2. 完成表单登记，上传License源文件
  3. 服务生成 License 后会自动触发浏览器下载：License.shield文件。

# 二、软件集成 License

# 2.1 引入Maven依赖

在你的工程中引入 smart-license 客户端依赖。

<dependency>
    <groupId>org.smartboot.license</groupId>
    <artifactId>license-client</artifactId>
    <version>2.0</version>
</dependency>
